The small print that saves time

This is a compiler, not a database.

01

Schema mapping matters

SQL names map to real Excel headers. M reads the display headers and table names exactly as Excel stores them.

02

Small subset, clear errors

CTEs, window functions, LIKE, LIMIT, and a few other constructs are rejected with a reason and a direction.

03

Excel is the runtime

The site produces source code. Paste it into Power Query and let Excel refresh the result against your workbook.

For Claude Code users

Same compiler, as an agent skill.

Drop skills/sql-to-power-m/ into your own project and an agent can compile SQL to Power Query M, or get a join-fan-out simplification suggestion, straight from the CLI — no browser, no upload.
